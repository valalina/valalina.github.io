Android Mobile Game

Croco Desert
Java / Android Studio

"CrocoDesert"

players swipe the crocodiles, which grab onto the food items in a slingshot-like motion. Through multithreading the waterfall items work independently. The Gameloop and Canvas provide a smooth gaming experience. Levels, Lives and High Scores are saved within Room Persistance Library.

Computer Graphics

PC Game
OpenGL / C++ / Blender

"TipBiker"

players have to guide George, a delivery biker to a randomly selected target city block within 30 seconds. George and the city blocks are designed in Blender. The game offers PhysX library collision detection, advanced graphics and realistic lighting. A skybox gives the illusion of a immersive world.

Information Visualization

web 3D presentation via
WebGL / three.js / d3.js

Data Visualization

Direct Volume Rendering on the GPU, raycasting and visualization of the volumetric data give the user a visual representation. A Density Histogram helps selecting iso-values for First-Hit Compositing and an interactive editor enables users to change the surface colors of the volumes.

Pattern Recognition

Automatic Reconstruction of
torn pieces of paper in Matlab

A4 Reconstruction

a MATLAB project to reconstruct scanned torn pieces of an A4 paper. Using image processing techniques like connected component labeling, Hough transformation, distance transformation, and matching algorithms, this tool seamlessly pieces together torn fragments.
